Supporting Files for Princeton Instruments’ PICam Library - picam_types¶

Defines data types, enums and some structures to be used with the main module picam!

As there are no enums in python, I have implemented each enumeration as dictionary having the same name and entries as the library version. In addition, for each enum XX, there is a lookup table called XXLookup, which takes the enum value as key and returns the name of the entry. For a complete list of valid enumeration values, please refer to the PICam SDK manual.
